# VonoaWeb — Sitio oficial (recreación dark-space)

Sitio completo recreado en un solo HTML, estilo "dark-space" del brand kit, con las secciones reales de vonoaweb.com:

- Hero sutil con cursor interactivo (starfield + spotlight)
- ¿Por qué elegirnos?
- Servicios (E-commerce, SEO, Diseño web, Chatbots) con las ilustraciones reales
- Beneficios de tener una página web
- Visión / Misión / Valores
- "Nuestra página web ofrece" (checklist)
- Precios (Impulso / Presencia Pro / Premium)
- Portafolio (Moncatu, Azul de Metileno, etc.) — imágenes reales de vonoaweb.com
- "¿Buscas una página web en Guadalajara?" CTA band
- Testimonios (Luisa, Eric, Jorge)
- Formulario de contacto
- Footer + botón flotante de WhatsApp

Las imágenes se cargan directamente desde `https://vonoaweb.com/wp-content/uploads/...` porque WordPress las sirve públicamente. Si prefieres alojarlas localmente avísame.
